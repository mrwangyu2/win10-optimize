# Windows 10 优化工具

这是一个简单而强大的 Windows 10 系统优化工具，旨在通过禁用不必要的服务、调整更新策略以及优化网络配置来提升系统性能。

## 功能特性

- **服务优化**：一键禁用不必要的后台服务，释放系统资源。
- **更新策略**：自定义 Windows 更新的最大暂停天数，由您掌控更新节奏。
- **网络优化**：
  - 解除系统保留带宽限制 (QoS)。
  - 根据内存大小自动优化入站 TCP 吞吐量级别。
- **单文件运行**：支持打包为单文件 `.exe`，绿色免安装。

## 快速开始

### 方案 A：直接使用 (推荐)
如果你是普通用户，建议直接从 GitHub 的 Releases 页面下载最新的 `.zip` 压缩包。解压后右键点击 **Win10优化工具.exe** 并选择“以管理员身份运行”即可，无需安装 Python 环境。

### 方案 B：源码运行 (开发者)
1. 确保已安装 Python 3.7+。
2. 克隆仓库或下载源代码。
3. 在项目根目录运行 `pip install -r requirements.txt` (本工具目前主要使用标准库)。
4. 双击运行 `run.bat` 或在终端执行 `python main.py`。

## 使用说明


1. 以管理员权限运行本程序。
2. 按照界面引导，依次配置服务、更新策略和网络选项。
3. 在“优化总结”界面确认任务后，点击“执行优化”。
4. 执行完成后，查看实时日志确认结果。
